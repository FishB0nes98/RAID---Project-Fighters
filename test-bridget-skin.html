How It Works

When Bridget has the Atlantean Ghost skin selected:

  1. The isBridgetAtlanteanGhost(character) function detects the skin
  2. The applyAtlanteanGhostSkin(element, character) function adds the CSS class
  3. CSS overrides automatically change all effect colors to spectral blue-green
  4. No VFX code rewriting needed - just CSS class application
